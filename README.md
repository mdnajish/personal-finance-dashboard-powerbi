**Personal Finance Dashboard – Power BI**

**Project Overview**

This project is an interactive Personal Finance Dashboard built using Microsoft Power BI to analyze income, expenses, and savings patterns.

The dataset was first cleaned and prepared using Microsoft Excel, then transformed and visualized in Power BI to generate meaningful financial insights.

This dashboard helps users track spending behavior, understand financial trends, and make better budgeting decisions.

**Project Objectives**

Track income and expenses

Analyze monthly spending patterns

Identify major expense categories

Identify major saving categories

Monitor savings and financial balance

Provide insights for better financial planning

**Project Tools**

**1️ Microsoft Excel**

Data cleaning

Data preparation

**2️ Microsoft Power BI**

Data modeling

DAX measures

Data visualization

Interactive dashboard creation

**Dashboard Features**

Total Income, Total Expenses, Savings and Net Worth KPIs

Monthly expense analysis

Category-wise spending distribution

Income vs Expense comparison

Interactive filters and slicers
